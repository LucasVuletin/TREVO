# TREVO Landing

Landing page institucional para TREVO, desarrollada con React, Vite y Tailwind CSS.

## Requisitos

- Node.js 18 o superior
- npm

## Desarrollo

```bash
npm install
npm run dev
```

## Build de producción

```bash
npm run build
```

## Despliegue en Vercel

No requiere configuración adicional. Vercel detecta Vite automáticamente:

- Build command: `npm run build`
- Output directory: `dist`

## Personalización rápida

Los links de contacto se pueden editar en:

- `src/data/siteContent.js`

Ahí están definidos:

- `WHATSAPP_URL`
- `INSTAGRAM_URL`
- `LINKEDIN_URL`
- `EMAIL`
